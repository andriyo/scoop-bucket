# ShadowDroid Scoop Bucket

Install ShadowDroid on Windows with Scoop:

```powershell
scoop bucket add andriyo https://github.com/andriyo/scoop-bucket
scoop install shadowdroid
```

Upgrade later:

```powershell
scoop update shadowdroid
```

ShadowDroid releases are published from:

https://github.com/andriyo/ShadowDroid/releases
